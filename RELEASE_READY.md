# StudyMate 打包完成说明

## ✅ 已完成

应用已经成功构建！你可以直接运行以下文件：

```
d:\studymate\release\win-unpacked\StudyMate 智学助手.exe
```

## 📦 应用内容

`win-unpacked` 目录包含完整的应用：
- StudyMate 智学助手.exe（主程序）
- 所有必需的 Electron 运行时文件
- 前端资源文件
- Python 后端（位于应用内部）

## 🚀 使用方法

### 直接运行（推荐）
1. 进入目录：`d:\studymate\release\win-unpacked\`
2. 双击 `StudyMate 智学助手.exe`
3. 应用将自动启动

### 创建桌面快捷方式
1. 右键点击 `StudyMate 智学助手.exe`
2. 选择"发送到" → "桌面快捷方式"
3. 以后双击桌面图标即可启动

### 创建便携版（可选）
如果你想创建一个单文件的可移植版本（不需要整个文件夹），可以使用 7-Zip 等工具将其压缩。

## ⚠️ 注意事项

1. **首次启动**：首次运行可能需要几秒钟来启动后端服务
2. **防火墙**：如果提示防火墙拦截，请允许访问
3. **端口占用**：确保端口 8000 未被其他程序占用
4. **数据保存**：用户数据保存在 `%APPDATA%\StudyMate\` 目录

## 🔧 故障排除

### 应用无法启动
- 检查是否所有文件都在同一目录
- 尝试以管理员身份运行
- 检查防火墙设置

### 后端服务无法启动
- 检查端口 8000 是否可用
- 查看应用目录下的日志文件

### 网络功能无法使用
- 确保网络连接正常
- AI 功能需要访问 DeepSeek API

## 📝 下次打包

如果以后需要重新打包，可以：
1. 打开 PowerShell
2. 进入项目目录：`cd d:\studymate`
3. 运行：`npm run electron:build:win`
4. 等待打包完成

## 💡 提示

- 你可以直接使用 `win-unpacked` 文件夹，无需安装
- 文件夹可以复制到任何位置（包括 U 盘）
- 删除整个文件夹即可完全卸载应用
