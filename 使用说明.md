# StudyMate 智学助手 - 使用说明

## 🚀 快速启动

### 方法一：开发模式（最简单，推荐）

1. 双击运行：`开发模式启动.bat`
2. 等待两个窗口都启动（需要几秒钟）
3. 在浏览器中访问：http://localhost:5173

### 方法二：手动启动

**第一步：启动后端**
1. 打开命令提示符或 PowerShell
2. 进入后端目录：`cd d:\studymate\backend`
3. 运行：`python main.py`

**第二步：启动前端**
1. 打开另一个命令提示符或 PowerShell
2. 进入前端目录：`cd d:\studymate\frontend`
3. 运行：`npm run dev`
4. 在浏览器访问：http://localhost:5173

## 📋 项目状态

✅ 前端已构建

## 💡 功能说明

- 📝 笔记管理：创建、编辑、删除笔记
- 📚 课程管理：管理课程分类
- 📊 闪卡学习：使用 AI 生成的闪卡复习
- 🎯 学习计划：AI 生成个性化学习计划
- 📖 文献助手：帮助润色和格式化文献

## ⚙️ 配置说明

- 数据库文件：`backend/data/studymate.db`
- AI API：已配置 DeepSeek API
- 后端端口：8000
- 前端端口：5173

## 📝 注意事项

1. 确保已安装 Python 和 Node.js
2. 首次运行会自动创建数据库
3. 两个终端窗口都要保持打开
4. 关闭窗口即可停止应用
5. 数据保存在本地，不会丢失
